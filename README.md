# Carbon Footprint Calculator for Products

 **Academic Year**: 2023 / 2024  
 **Technologies**: React (Frontend) + JSON Backend  
 **Project Type**: University Project  

---

##  Project Overview

This project is a **web-based application** designed to **calculate the carbon footprint of different products**.  
It helps users estimate environmental impact by analyzing product-related data stored in a structured database.

The application provides an intuitive interface where users can explore products, view their carbon emissions, and compare environmental impacts.

---

##  Project Objectives

- Raise awareness about **carbon emissions** and environmental impact
- Provide an easy-to-use tool for **carbon footprint calculation**
- Store and manage product data using a **JSON-based backend**
- Apply modern **web development practices** using React

---

##  Technologies Used

###  Frontend
- **React.js**
- HTML5
- CSS3
- JavaScript (ES6)

###  Backend / Data
- **JSON-based backend**
- Structured product database
- REST-style data access (local or simulated API)

---

##  Application Features

-  Product database management
-  Carbon footprint calculation for each product
-  Visualization of emission values
-  Product comparison
-  Modular React components
-  Fast and responsive user interface

---

##  Database Description

The backend database stores information such as:
- Product name
- Category
- Production data
- Emission factors
- Calculated carbon footprint values
